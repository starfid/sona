## Starfid Sona
- All purpose UI kit. Keyboard-first operations.
- One responsive layout for all: desktop, tablet, phone.
- Automatic dark mode support.
- Keyboard Arrows keys + Enter for navigation.
- Selection shortcut, Shift + Arrows, Del key.
- 2D Barcode and QRCode scanner.
- List/tiles view.
