## Starfid Sona
- All purpose ui kit.
- Responsive layout: desktop, tablet, phone.
- Responsive web page.
- Keyboard arrows navigation.
- Selection shortcut, Shift + Arrows, Del key.
- 2D Barcode, QRCode scanner.
- List/tiles view.
